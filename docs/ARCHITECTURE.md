## 运行流程

### 1. 启动流程

```
┌─────────────────────────────────────────────────────────────┐
│                     MnMCP 启动流程                          │
└─────────────────────────────────────────────────────────────┘

[1] 配置加载
    │
    ├── 读取 config.yaml
    ├── 加载服务器配置
    └── 初始化日志系统
    │
    ▼
[2] 加密初始化
    │
    ├── 加载 ECDH 密钥对
    ├── 初始化 HKDF
    └── 创建 AES-GCM 加密器
    │
    ▼
[3] 网络连接
    │
    ├── HTTPS 认证 (certification.mini1.cn)
    ├── 房间分配 (openroom.mini1.cn)
    └── TCP 连接 (game_server:4012)
    │
    ▼
[4] 协议握手
    │
    ├── 发送 LOGIN_REQ
    ├── 接收 LOGIN_RESP
    └── 激活 AES 加密
    │
    ▼
[5] 桥接启动
    │
    ├── 连接 Minecraft 服务器
    ├── 启动协议转发
    └── 开始玩家同步
    │
    ▼
[6] 游戏循环
    │
    ├── 接收 MNW 消息
    ├── 转换并转发到 MC
    ├── 接收 MC 消息
    └── 转换并转发到 MNW
```

├─ 位置 (X, Y, Z)       <───>        ├─ 位置 (X, Y, Z)

└─ 方块元数据           <───>        └─ 方块状态

### 4. 加密流程

```
┌─────────────────────────────────────────────────────────────┐
│                   加密流程示意图                             │
└─────────────────────────────────────────────────────────────┘

[1] 密钥交换 (ECDH)
    │
    ├── 客户端生成临时密钥对
    ├── 发送公钥到服务器
    ├── 服务器返回公钥
    └── 计算共享密钥
    │
    ▼
[2] 密钥派生 (HKDF)
    │
    ├── 输入: ECDH 共享密钥
    ├── 输出: 48字节密钥材料
    │   ├── AES Key: 16字节
    │   ├── Nonce Base: 12字节
    │   └── Padding: 20字节
    │
    ▼
[3] 数据加密 (AES-128-GCM)
    │
    ├── 输入: 明文数据
    ├── 密钥: AES Key
    ├── Nonce: Nonce Base + 计数器
    ├── 输出: 密文 + Auth Tag
    │
    ▼
[4] WPKG 打包
    │
    ├── 包头 (16字节)
    │   ├── Magic: 0xABCD
    │   ├── Version: 0x01
    │   ├── Flags: ENCRYPTED
    │   └── Sequence Number
    ├── 加密数据
    └── Auth Tag
```
