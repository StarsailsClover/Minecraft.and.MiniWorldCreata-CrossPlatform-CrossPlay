# MnMCP 两款游戏互联分析报告

**分析日期**: 2026-02-27  
**分析版本**: v0.1.0  
**分析状态**: 🔴 **无法实现互联 - 关键组件缺失**

---

## 执行摘要

经过详细分析，当前项目**无法实现 Minecraft 与 迷你世界 的真正互联**。

**根本原因**: 缺少以下4个关键组件：
1. Java中转服务器 (PaperMC)
2. GeyserMC桥接器
3. Floodgate身份映射
4. 真实迷你世界服务器连接

**当前状态**: 只有基础TCP代理框架，缺少协议转换核心。

## 详细差距分析

### 1. ❌ Java中转服务器 (优先级: 🔴 高)

**需要**: PaperMC 1.20.6 + Fabric
- 处理Minecraft协议
- 管理玩家会话
- 广播世界状态

**当前**: 只有简单的TCP代理
- 无法理解MC协议
- 无法处理游戏逻辑
- 只是透传数据

**影响**: ⭐⭐⭐⭐⭐ (致命)
- 无法让Minecraft客户端"进入游戏"
- 没有游戏世界可以加入

**解决方案**:
```bash
# 1. 下载 PaperMC
wget https://api.papermc.io/v2/projects/paper/versions/1.20.6/builds/latest/downloads/paper-1.20.6-latest.jar

# 2. 启动服务器
java -jar paper-1.20.6-latest.jar

# 3. 配置服务器属性
echo "online-mode=false" >> server.properties
echo "server-port=25566" >> server.properties
```

---

### 2. ❌ GeyserMC桥接器 (优先级: 🔴 高)

**需要**: GeyserMC插件
- Java ↔ Bedrock 协议转换
- 让基岩版玩家加入Java服务器

**当前**: 无
- 不支持基岩版
- 只有Java版协议

**影响**: ⭐⭐⭐⭐ (重要)
- 无法支持Minecraft基岩版
- 只能支持Java版玩家

**解决方案**:
```bash
# 1. 下载GeyserMC
wget https://download.geysermc.org/v2/projects/geyser/versions/latest/builds/latest/downloads/geyser.jar

# 2. 放入plugins目录
mv geyser.jar plugins/

# 3. 配置Geyser
echo "bedrock port: 19132" >> plugins/Geyser-Spigot/config.yml
```

---

### 3. ❌ Floodgate身份映射 (优先级: 🔴 高)

**需要**: Floodgate插件
- 基岩版玩家免Java账号登录
- 跨端身份映射

**当前**: 无
- 基岩版玩家无法认证

**影响**: ⭐⭐⭐⭐ (重要)
- 基岩版玩家无法登录
- 缺少跨端身份管理

**解决方案**:
```bash
# 1. 下载Floodgate
wget https://download.geysermc.org/v2/projects/floodgate/versions/latest/builds/latest/downloads/floodgate.jar

# 2. 放入plugins目录
mv floodgate.jar plugins/
```

### 5. ❌ 生产级加密模块 (优先级: 🟡 中)

**需要**: 真实AES-128-CBC / AES-256-GCM
- 解密迷你世界数据
- 加密发送数据

**当前**: 简化版XOR加密
```python
# 当前实现 (简化版)
def encrypt(self, data):
    # 使用XOR加密 (仅用于测试!)
    for i, byte in enumerate(data):
        key_byte = self.key[i % len(self.key)]
        encrypted.append(byte ^ key_byte)
```

**影响**: ⭐⭐⭐ (中等)
- 无法解密真实MNW数据
- 无法与真实服务器通信

**解决方案**:
```bash
# 安装cryptography库
pip install cryptography
```

```python
# 使用真实AES
from cryptography.hazmat.primitives.ciphers import Cipher, algorithms, modes

cipher = Cipher(algorithms.AES(key), modes.CBC(iv))
encryptor = cipher.encryptor()
encrypted = encryptor.update(data) + encryptor.finalize()
```

## 结论

### 🔴 当前状态: 无法实现互联

**原因**:
1. 缺少Java中转服务器
2. 缺少GeyserMC桥接
3. 缺少Floodgate身份映射
4. 缺少真实MNW连接
5. 加密模块不完整

### 📋 需要完成的工作

1. **搭建Java服务器** (2-3天)

2. **逆向MNW协议** (2-3周)
