## 使用说明

### 快速启动

```bash
start_test.bat
```

### 交互式测试

```bash
interactive_test.bat
```

或 PowerShell:

```powershell
.\interactive_test.ps1
```

### 真实连接测试

```bash
python real_connection_test.py
```

### 参数化启动

```powershell
# 完整测试
.\interactive_test.ps1 -FullTest

# 仅启动桥接器
.\interactive_test.ps1 -StartBridge

# 测试数据流
.\interactive_test.ps1 -TestDataFlow

# 启动Wireshark
.\interactive_test.ps1 -StartWireshark

# 启动迷你世界
.\interactive_test.ps1 -StartMiniWorld

# 查看日志
.\interactive_test.ps1 -ViewLogs
```
