## 资源需求

### 开发环境
- Java 17+ (PaperMC需要)
- Python 3.11+
- 16GB+ RAM
- SSD存储

### 测试环境
- Minecraft Java 1.20.6
- Minecraft Bedrock (Windows 10版)
- 迷你世界 PC版 1.53.1
- 网络抓包工具 (Wireshark)

- 1名全栈开发者 (当前)

- 可选: 1名逆向工程师 (MNW协议)
